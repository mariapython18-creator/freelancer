# Freelance Task Marketplace

## Overview
This is a fully functional freelance task marketplace web application built with **Python and Django**. It allows clients to post jobs, receive submissions from freelancers, make payments, and manage projects. Freelancers can explore jobs, submit their work, and apply for projects. The platform also includes tutorials to help freelancers improve their skills.

---

## Features

### **Client Features**
- Post new jobs/tasks with details and deadlines
- View applications from freelancers
- Accept submissions from freelancers
- Make payments to freelancers via integrated payment gateway
- Track job progress and status

### **Freelancer Features**
- Browse and apply for available jobs/tasks
- Submit completed projects for client approval
- View payment history and completed projects
- Explore tutorials and resources for skill development

### **Admin Features**
- Add, update, and delete job categories
- Manage users (clients and freelancers)
- Manage posted jobs and submissions
- Dashboard with statistics (optional)

### **Common Features**
- Full CRUD operations for jobs, projects, and users
- Payment integration (test mode enabled)
- Responsive and user-friendly UI

---

## Technology Stack
- **Backend:** Python, Django
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Database:** SQLite / MySQL (as configured)
- **Payment Gateway:** Razorpay
